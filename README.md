# Gulptrap startup template
A [Bootstrap](https://getbootstrap.com/) v4.1.3 startup template with [Sass](http://sass-lang.com/), concatenation, minification, autoprefixer, [Browsersync](https://www.browsersync.io/), hot reloading and sourcemaps all running by [Gulp](https://gulpjs.com/).
